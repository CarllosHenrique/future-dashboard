# DASHBOARD

### Commit and Versioning for big features
```sh
git checkout -b [branch]
git add .
git commit -m "[DSH]:[X.0.0]: description"
git push
```

### Commit and Versioning for any features
```sh
git add .
git commit -m "[0.x.x]:[method]: description"
git push
```
